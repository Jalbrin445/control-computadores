# Sistema Gestión de Mantenimiento de Equipos de Computo

## Descripción del proyecto

Este es un sistema de escritorio que facilita la gestión de equipos de computo en estado de mantenimiento o reparación, realmente es un sistema sencillo que facilita algunas funciones como las CRUD y MVP de una tienda o taller dedicado a brindar estos servicios a diferentes clientes o ciudadanos, es por ello que este proyecto que nacio de una necesidad educativa lo hiré desarrollando poco a poco con el fin de lanzar varias versiones hasta tener un sistema totalmente sofisticado que posteriormente implementaré en versión web con Flask, FastAPI, Django u otro framework o lenguaje de programación como JavaScript con React, Vue.js o Angular y Node.JS (Con Express o cualquier otro backend), con el fin de tener varias versiones.

## Dependencias de este sistema en distintas versiones.

1. Versión Sencilla.

* Tkinter: librería utilizada para implementar interfaces de usuario de forma agil en sistemas de escritorio con gran facilidad de desarrollar software MVP.

* Datatime: librería utilizada para la gestión de hora con diferentes modulos, métodos y propiedades que permiten formatear horas, minutos, días e incluso obtener horas en tiempo real. 

### Para instalar las dependencias:

* pip install requeriments.txt (NOTA: en estas primeras versiones como se estan utilizando librerías que ya vienen preinstaladas en el paquete de python entonces no son necesarias volverlas a instalar)
